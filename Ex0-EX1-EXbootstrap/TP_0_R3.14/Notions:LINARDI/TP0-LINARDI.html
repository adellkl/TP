  • BROWSERENGINE :

    • Un navigateur Web est une application logicielle permettant d'accéder à des informations sur Internet.
    Chaque page Web, image et vidéo individuelle est identifiée par une URL (Uniform Resource Locator) distincte, permettant aux navigateurs de récupérer ces ressources à partir d'un serveur Web et de les afficher sur l'appareil d'un utilisateur. Lorsque vous choisissez un navigateur Web, vous choisissez également un moteur de navigation et un moteur de rendu - parfois ces deux termes sont utilisés séparément et parfois pour signifier la même chose.

    -Interpretation professionnelle :

    un moteur de navigateur en français est un moteur de mise en page ou ( un moteur de rendu ) il permet d'afficher le contenu c'un fichier html css php javascript... , son rôle principale est d'afficher le code d'une façon lisible et comprehensible par l'utilisateur .


  • RENDERINGENGINE :

    • Un moteur de rendu est un logiciel qui dessine du texte et des images à l'écran. Le moteur extrait du texte structuré d'un document (souvent HTML ) et le formate correctement en fonction des déclarations de style données (souvent données en CSS ). Exemples de moteurs de mise en page : Blink , Gecko , EdgeHTML, WebKit .

  • NETWORKING

    • Avant de définir le networking, il est important de savoir de quoi on parle. Cette pratique revêt plusieurs noms qui traitent du même sujet.
    Réseautage ou réseautage d’affaires sont les synonymes les plus courants que tu pourras entendre ou même utiliser pour parler de networking.
    En résumé, la définition de networking est simple et presque transparente : il s’agit de constituer un réseau, souvent professionnel, en nouant des relations avec des personnes.
    L’objectif ? Développer des contacts qui pourront t’apporter des opportunités professionnelles ou à qui tu pourras faire appel pour d’éventuels partenariats. Le networking n’est pas toujours une pratique dont le but est d’en tirer quelque chose. Certains nouent ce réseau tout simplement pour développer des connaissances qui travaillent sur des sujets divers et étendre leurs connaissances sur ces mêmes sujets.
    C’est pour cela que l’on identifie deux types de pratiques : le social networking, dont l’objectif est plutôt de se faire des amis, et le business networking, qui a plutôt pour but de nouer des relations professionnelles.

  • JAVASCRIPTINTERPRETER :

    • Un moteur JavaScript est un programme logiciel qui interprète et exécute du code en langage JavaScript. Les moteurs JavaScript sont généralement intégrés aux navigateurs Web.
    Le premier moteur JavaScript, SpiderMonkey, a été créé par l'informaticien américain Brendan Eich pour le navigateur Netscape Navigator.

    -Interpretation professionnelle :

    JavaScript est un langage de programmation et un editeur de code
